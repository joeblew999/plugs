# us-task-ui

Web GUI for Taskfile.yml - run tasks from your browser.

**Docs:** [User Guide](us-task-ui_user.md) Â· [Technical](us-task-ui_tech.md)

## Usage

```sh
# Start web UI on default port
us-task-ui

# Specify port
us-task-ui --port 8080
```

Then open http://localhost:8080 in your browser.

## Install

```sh
plugctl install us-task-ui
```

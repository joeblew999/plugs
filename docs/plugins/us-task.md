# us-task

Task runner for Taskfile.yml automation.

**Docs:** [User Guide](us-task_user.md) Â· [Technical](us-task_tech.md)

## Usage

```sh
# List available tasks
us-task --list

# Run a task
us-task build

# Run with arguments
us-task build -- --verbose
```

## Install

```sh
plugctl install us-task
```
